# 📰 Fake News Detection using Machine Learning

## 📌 Overview
This project predicts whether a news article is **Fake** or **Real** using Machine Learning.

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask
- HTML
- CSS

## 🤖 Machine Learning Algorithm
- TF-IDF Vectorizer
- Logistic Regression

## 📊 Accuracy
99%
